# Hi bro this is like a roadmap trial thing

### how to run
- run `pip install flask streamlit google-generativeai python-dotenv requests`
- make a .env file and in it put `GEMINI_API_KEY=your_api_key_here` (your gemini key should replace the placeholder)
- run `python backend.py` in one terminal
- run `streamlit run frontend.py` in another terminal
